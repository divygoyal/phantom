## §X. India-targeting tactics (Hinglish reels)

Apply when `audience_persona_lean ∈ {AI-curious-IN, AI-builder-IN}` AND topic warrants Hinglish density >30%.

- **Hinglish 42% better engagement** than pure English in Tier 2/3 Indian cities.
- **3-second hook window**, fail = algorithm flags low-relevance.
- **Fast J-Cuts every 0.8s** — sub-second cut rate (overrides the standard 1.5-2.5s cut cadence in retention-template.md).
- **85% of Indian social video watched MUTED** — burn-in captions mandatory. Critique auto-fail #17 enforces.
- **Code-mix density**: ~60% Hindi / 40% English. English for technical/business terms; Hindi for emotional emphasis.
- **Bollywood "Interval Twist" at second 2** — premise flip exactly at t=2s.
- **Always-end-with-a-song closer** (Aevy TV signature) — 2-sec branded music sting.
- **Font for code-mix**: Mukta (EkType, free, Devanagari + Latin, shares x-height with Inter).

### Hinglish anti-patterns (do NOT)

- **Don't translate tool names** ("Bina-Banana Pro" instead of "Nano Banana Pro") — kills SEO.
- **Don't use hyper-formal Hindi** (Sanskritized vocabulary) — reads off-channel; this audience speaks casual Hinglish.
- **Don't write English script and ask voice talent to "do it Indian"** — reads as mockery; full Hinglish or full English.
- **Don't use Mukta on US-targeted reels** — reads off-channel.

### When NOT to use Hinglish

If `audience_persona_lean = mixed-default` AND topic is global (model launch, frontier capability), default to English-only. Hinglish on a global topic reads as audience-segmenting and reduces unconnected reach.

## §X.1 Specifically @aisimplified moves

Cross-reference `audience-aisimplified.md` for the persona. The channel-specific moves:

- **Use @aisimplified handle in EVERY CTA / comment-mockup / social-handle slot.** Locked-in. Critique flags if "@yourchannel" appears.
- **Default to save-CTA**, with soft keyword fallback for IN audiences ("save this; if you want the link, drop a word").
- **Always name the tool in the hook** when topic is launch / feature / news. Nano Banana, Sora 2, Claude Code, Veo 3.1 — naming gates SEO + cohort-routing.
- **Specific numbers** in hooks outperform vague claims by ~2× ("4 seconds", "100M users", "$0 vs $200/mo").
- **NO course-bro hype words** in scripts — "incredible / mind-blowing / game-changing / next-level". Critique auto-fail.
- **Cooper italic + Inter caption stack default** for premium register (sincere-awe, ironic-deadpan). Mukta when Hinglish density >30%. NEVER Hormozi yellow on these audiences.
